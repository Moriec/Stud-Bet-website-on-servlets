# Stud-Bet - Платформа студенческих ставок

Веб-приложение для организации ставок среди студентов на основе Java Servlets. Проект разработан в рамках семестровой работы.

## Описание

Stud-Bet - это платформа, которая позволяет студентам делать ставки на различные события. Система включает в себя регистрацию пользователей, управление балансом, историю транзакций и административную панель.

## Основные возможности

- Аутентификация и авторизация - регистрация и вход пользователей с хешированием паролей (BCrypt)
- Система ставок - расчет коэффициентов и возможность делать ставки
- Профиль пользователя - просмотр баланса и личной информации
- История транзакций - отслеживание всех финансовых операций
- Рейтинговая система - таблица лидеров среди пользователей
- Административная панель - управление предметами и событиями для ставок
- Страницы правил и FAQ - информация для пользователей
- Защита от XSS атак - безопасность приложения

## Технологический стек

- Backend: Java 8, Java Servlets 4.0
- Frontend: FreeMarker 2.3.34
- База данных: PostgreSQL 15
- Connection Pool: HikariCP 7.0.2
- Безопасность: jBcrypt 0.4
- Валидация: Apache Commons Validator 1.8.0
- Сборка: Maven 3
- Контейнеризация: Docker, Docker Compose
- Сервер приложений: Apache Tomcat 9.0

## Установка и запуск

### Требования

- Docker и Docker Compose
- Порты 8080 и 5432 должны быть свободны

### Запуск через Docker Compose

```bash
# Клонировать репозиторий
git clone https://github.com/Moriec/Stud-Bet-website-on-servlets.git
cd Stud-Bet-website-on-servlets

# Запустить приложение
docker-compose up --build
```

Приложение будет доступно по адресу: http://localhost:8080

## База данных

Структура базы данных автоматически создается при первом запуске через Docker Compose. SQL скрипт находится в flyway-migration/create_bd.sql.

### Основные таблицы:

- users - пользователи системы
- bets - ставки пользователей
- transactions - история финансовых операций
- subjects - предметы для ставок
- events - события для ставок
